## 📝 프로젝트 소개

집주인 직거래 및 중개사 협의 가능한 제주 곶자왈아이파크 32평 급매 매물을 소개하는 웹사이트입니다.

## ✨ 주요 특징

- **💰 급매 조건**: 시세 대비 저렴한 가격 (매매 5억 / 전세 4억 3천)
- **📍 최고의 입지**: 제주 국제학교 통학버스 이용 가능, 곶자왈 숲 인근
- **🏢 프리미엄 시설**: 호텔급 커뮤니티, 조식 서비스, 지하주차장
- **🎬 풍부한 콘텐츠**: 35장의 고화질 사진 + 실제 매물 투어 동영상

## 🖼️ 갤러리 구성

- **거실**: 7장 (다양한 각도의 넓은 거실)
- **주방**: 4장 (모던한 주방과 수납공간)
- **침실/방**: 5장 (메인 침실 + 다용도 방)
- **화장실/파우더룸**: 5장 (깔끔한 디자인)
- **발코니/베란다**: 3장 (바베큐 가능한 공간)
- **편의시설**: 5장 (아파트 커뮤니티 시설)
- **평면도**: 4장 (상세한 구조 확인)

## 🚀 기술 스택

- **HTML5**: 시맨틱 마크업
- **Tailwind CSS**: 반응형 디자인
- **JavaScript**: 이미지 슬라이더, 동영상 최적화
- **반응형**: 모바일/태블릿/데스크톱 최적화

## 🌐 배포

이 사이트는 GitHub Pages를 통해 배포됩니다.

### 로컬 실행 방법

1. 저장소 클론: `git clone [저장소 URL]`
2. `index.html` 파일을 브라우저에서 열기

### GitHub Pages 배포

1. GitHub 저장소의 Settings 탭으로 이동
2. Pages 섹션에서 Source를 "Deploy from a branch" 선택
3. Branch를 "main" (또는 "master") 선택
4. Save 버튼 클릭
