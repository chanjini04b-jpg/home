# SNS 공유 및 검색 최적화 가이드

## 🔗 SNS 공유 기능

### 지원하는 플랫폼

- **카카오톡**: 한국 사용자를 위한 주요 공유 수단
- **페이스북**: 글로벌 SNS 플랫폼
- **트위터**: 실시간 정보 공유
- **WhatsApp**: 국제 메신저 앱
- **링크 복사**: 모든 플랫폼에서 사용 가능
- **QR 코드**: 모바일 빠른 접근

### 사용 방법

1. 웹사이트 하단의 "매물 정보 공유하기" 섹션으로 이동
2. 원하는 SNS 플랫폼 버튼 클릭
3. 자동으로 매물 정보와 이미지가 포함된 게시물 생성

## 🔍 검색 엔진 최적화 (SEO)

### 현재 적용된 최적화

#### 1. 메타 태그 최적화

```html
- 페이지 제목: "제주 곶자왈아이파크 32평 급매" - 설명: 매물의 핵심 정보 포함 -
키워드: 제주부동산, 곶자왈아이파크, 급매 등
```

#### 2. Open Graph (소셜 미디어 최적화)

- 제목, 설명, 이미지가 자동으로 SNS에 표시
- 카카오톡, 페이스북 등에서 미리보기 제공

#### 3. 구조화된 데이터 (JSON-LD)

```json
- Schema.org의 RealEstateListing 타입 사용
- 주소, 가격, 연락처 등 구조화된 정보 제공
- Google 검색에서 리치 스니펫 표시 가능
```

#### 4. 사이트맵 (sitemap.xml)

- 검색 엔진이 사이트 구조를 이해할 수 있도록 도움
- 이미지와 동영상 정보도 포함

#### 5. Robots.txt

- 검색 엔진 크롤러에게 크롤링 규칙 제공
- 중요한 콘텐츠 우선 인덱싱 지시

## 🌐 검색 가능성 높이는 방법

### 1. 배포 후 즉시 할 일

#### Google Search Console 등록

1. [Google Search Console](https://search.google.com/search-console/) 접속
2. 속성 추가: 웹사이트 URL 입력
3. 소유권 확인 (HTML 파일 업로드 또는 메타 태그)
4. sitemap.xml 제출

#### 네이버 서치어드바이저 등록

1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
2. 사이트 등록 및 소유권 확인
3. 사이트맵 제출

#### 다음 검색등록

1. [다음 검색등록](https://register.search.daum.net/index.daum) 접속
2. 사이트 URL 직접 등록

### 2. 소셜 미디어 활용

#### 카카오톡 채널/그룹

- 부동산 관련 카카오톤 그룹에 공유
- 지역 커뮤니티 그룹 활용

#### 페이스북 그룹/페이지

- 제주 부동산 관련 그룹 게시
- 개인 페이지에서 공유

#### 인스타그램

- 매물 사진을 인스타그램에 업로드
- 해시태그 활용: #제주부동산 #곶자왈아이파크 #급매

#### 네이버 블로그/카페

- 네이버 블로그에 매물 정보 포스팅
- 부동산 관련 카페에 글 작성

### 3. 백링크 구축

#### 부동산 사이트 등록

- 직방, 다방 등 부동산 플랫폼 (유료)
- 네이버 부동산, 다음 부동산

#### 지역 커뮤니티

- 제주 지역 커뮤니티 사이트
- 동네 카페, 맘카페 등

#### 블로그/포럼

- 부동산 투자 블로그 소개글
- 제주 이주 관련 포럼

## 📊 성과 측정

### Google Analytics 설정 (권장)

```html
<!-- Google Analytics 코드를 <head>에 추가 -->
<script
  async
  src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"
></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag() {
    dataLayer.push(arguments);
  }
  gtag("js", new Date());
  gtag("config", "GA_MEASUREMENT_ID");
</script>
```

### 추적할 수 있는 지표

- **방문자 수**: 일별/주별 방문자 현황
- **페이지뷰**: 가장 인기있는 섹션 파악
- **체류시간**: 사용자 관심도 측정
- **유입 경로**: 어디서 방문하는지 파악
- **모바일/데스크톱 비율**: 디바이스별 사용 현황
