## ⚠️ 현재 이미지 최적화 상태:

**개선 필요**: 현재 이미지들이 평균 8-9MB로 웹용으로는 큰 편입니다.

### 권장 최적화 방법:

1. **파일 크기 압축**: 1-3MB로 압축 권장
2. **WebP 포맷 변환**: 더 효율적인 압축 (30-50% 크기 절약)
3. **해상도 조정**: 웹용 최적 해상도로 조정

## 권장 이미지 사양:

- **해상도**: 1200x800 픽셀 이상
- **파일 형식**: JPG, PNG 또는 WebP (권장)
- **파일 크기**: 1-3MB 권장 (현재 8-9MB)
- **화질**: 고화질 (선명하고 밝은 사진)

## 촬영 팁:

1. **거실 사진**: 넓어 보이도록 코너에서 대각선으로 촬영
2. **주방 사진**: 깔끔하게 정리된 상태에서 전체적인 구조가 보이게
3. **베란다 사진**: 바깥 전망과 함께 공간의 넓이를 강조
4. **외관 사진**: 아파트 건물 전체와 주변 조경을 포함
